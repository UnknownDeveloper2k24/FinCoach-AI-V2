## ğŸ“ File Naming Conventions

### Backend

- **Classes**: `PascalCase` (e.g., `User`)
- **Functions**: `snake_case` (e.g., `get_user()`)

- **Pages**: `page.tsx` in directory
- **Hooks**: `useXxx.ts` (e.g., `useTransactions.ts`)
